1st century before Christ

The Caesar code is the first real encryption code. It has been created by Roman army itself, to secure the communications between Caesar and his allies on the first century BC and to secure the message if it was somehow stolen by ennemies. We will explain you that method later on.

1467

On 1467, an italian named Alberti created a polyalphabetic substitution cipher disk which changes letters of the message into the same, but in differents alphabets. Furthermore, he was the first man who created a super-encryption (encrypting a message already encrypted).

XVIth century

The hiding method of steganography was discovered on 1518. It's the way to hide sentences into a whole text : if you read one line out of two, the first word of each sentence or each line and really more complex ways. This method was still used during the wars by soldiers to hide messages in their letters to communicate something secret to their families or friends.
Blaise de Vigenère was a french man, who created the Vigenère code on 1586. Finally, this code has been broken on 1854. We will explain you how it works in the method part.

1623

In 1623, Francis Bacon created a encryption code representing each letter by a group of 5 letters, composed of A and B (AAAAA= A / AAAAB = B). It can be cinsidered the ancestor of binary, which is still used in computer science.

1832

In 1832, the morse code was created, it's a method where each letter or number is represented by an unic composition of lines and dots, used by telegraph, armies, etc... This encryption is one of the most widely known, therefore not secure at all, but it can be quite useful because it is possible to communicate in morse simply and discreetely, such like blinking or tapping your thigh with two fingers.

End of the XIXth century

In the end of the XVIIIth century and the start of the XIXth century, there is the first using of a coded transmition with the optical telegraph of Chappe(french revolutionaire). It's a succession of stations with arms on the top pf the building and is send to the reciever. According to the position of arms, we have a letter. It was on 1794. Also, it has been used by the twins Joseph and François Blanc between 1834 and 1835

1939-1945

During the World War Two, Enigma, a coding machine with some rotors, has been created by the german army to hide the time and location of their attack. Alan TURING decoded Enigma with a computer created by himself, he found one word which was everytime in german messages (it was the name of the reichfurer, Hitler). We will develop this operation in the method part.
Lorenz's Number was another machine used during World War Two, to encrypt teleprinter.

1949

Claude Shannon was considered like the father of the modern cryptology. He showed, in 1949, that to have a perfect encryption, we need an encryption key as long as the message himself.

1975

The Data Encryption Standard (DES) was the first encryption code accessible to the masses, on 1975.

Today

Nowadays, a lot of powerful encryption codes are availables and non governemantals organisms can get them. Governments don't agree with this method of hiding information because they can't decrypt those encryptions. The conflict already exists and is called Crypto-War.
